## Get started
1. Install dependencies
 
   ```bash
   git clone https://github.com/noNameDS09/FarmVichar.git
   ```

1. Install dependencies
 
   ```bash
   cd FarmviChar
   ```

3. Install dependencies

   ```bash
   npm install #or npm i
   ```

2. Start the app

   ```bash
   npx expo start
   ```

- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
